## 一、模块概述

### 1.1 背景与目标

交易所后台管理系统包含多个业务模块（数据中心、系统管理、财务中心、客服中心、运营中心），管理员日常工作中频繁切换模块的场景十分常见。

本模块目标：
- 提供**模块入口聚合页**，支持按使用热度排列模块，最常用的模块最醒目
- 支持**模块入口弹窗**快速切换，无需记忆菜单路径
- 动效与交互体验精细化，提升后台操作效率与愉悦感

### 1.2 术语定义

| 术语 | 定义 |
|---|---|
| 模块入口页 | 登录后展示的模块选择/快捷入口页面（热力图布局） |
| 模块入口弹窗 | 顶部导航栏"更多"按钮触发的模块快速切换浮层 |
| 热力图布局 | 按模块使用频率分配不同尺寸区域的响应式布局 |
| 使用频次 | 管理员点击某模块的累计次数，本地缓存 |

## 二、模块入口页（主入口）

### 2.1 布局规格

| 属性 | 值 |
|---|---|
| 模块入口区域最大宽度 | 1100px |
| 水平居中 | 是 |
| 背景色 | `#0f0f0f`（深色主题，与后台整体风格统一） |
| 模块卡片圆角 | 12px |
| 模块卡片间距 | 16px |

**热力图布局规则：**
- 面积最大的模块（使用频次最高）位于**左侧**
- 面积次大的模块依次向右排列
- 支持**反向排列**开关（最常用显示在右侧），本地缓存偏好
- 区域宽度比例：高频模块宽，低频模块窄（宽度比例按频次对数映射）

### 2.2 模块卡片

每个模块卡片包含：
- **模块图标**：SVG格式，48x48px（大方块）/ 36x36px（中）/ 24x24px（小）
- **模块名称**：中文名称，如"数据中心"、"系统管理"
- **模块描述**（可选）：一行简介，默认隐藏，hover时显示

**卡片尺寸分级：**
| 级别 | 宽度占比 | 高度 | 最小点击区域 |
|---|---|---|---|
| 一级（最常用） | 50%（1100px区域） | 120px | 200x120px |
| 二级 | 25% | 120px | 150x120px |
| 三级 | 25% | 120px | 150x120px |
| 四级及以下 | 各自均分剩余空间 | 100px | 120x100px |

**一行最多4个模块，总高度自适应。**

### 2.3 动效规格

| 交互 | 动效描述 | 参数 |
|---|---|---|
| 鼠标移入 | 卡片上浮 + 边框发光 + 图标缩放 | transform: translateY(-6px)，transition: 0.3s cubic-bezier(0.34, 1.56, 0.64, 1)，box-shadow扩散 |
| 鼠标移出 | 平滑回落 | transition: 0.2s ease-out |
| 页面加载 | 模块卡片依次从下往上淡入 | stagger 50ms，duration 400ms |
| 排列切换 | 卡片整体平滑滑动 | transition: 0.5s ease |

### 2.4 本地缓存

使用 `localStorage` 缓存以下信息：

```json
{
  "moduleEntryPrefs": {
    "layoutDirection": "left-heavy" | "right-heavy",
    "usageStats": {
      "data-center": 45,
      "system-admin": 30,
      "finance": 20,
      "customer-service": 15,
      "operations": 10
    },
    "lastUpdated": 1713254400000
  }
}
```

- **layoutDirection**：排列方向，`"left-heavy"`（默认）或 `"right-heavy"`
- **usageStats**：各模块点击次数，每次进入任意模块时 +1
- **缓存更新**：每次点击模块时异步更新，不阻塞导航
- **缓存过期**：90天无操作则重置usageStats

## 三、顶部导航栏入口

### 3.1 布局

登录成功后，页面顶部导航栏结构：

```
[Logo]  [Dashboard/首页]  [导航菜单...]  [更多▼]  [头像/用户名]  [设置]
```

- Logo左侧，导航菜单右侧
- "更多"按钮显示向下箭头图标
- 按钮宽度自适应文字，padding: 8px 12px

### 3.2 更多按钮样式

| 属性 | 值 |
|---|---|
| 字体大小 | 14px |
| 颜色（默认） | `#8c8c8c` |
| 颜色（hover） | `#ffffff` |
| 圆角 | 6px |
| 背景（hover） | `rgba(255,255,255,0.1)` |

### 3.3 点击行为

1. 点击"更多"按钮 → 展开**模块入口弹窗**
2. 再次点击"更多"按钮或点击页面其他区域 → 关闭弹窗

## 四、模块入口弹窗

### 4.1 弹窗规格

| 属性 | 值 |
|---|---|
| 弹窗宽度 | 600px |
| 弹窗最大高度 | 480px（超出滚动） |
| 背景色 | `#1a1a1a` |
| 弹窗圆角 | 12px |
| 遮罩层颜色 | `rgba(0,0,0,0.6)` |
| 弹出动画 | 从"更多"按钮位置向下扩展，scale + opacity，200ms |

### 4.2 弹窗内容

弹窗内以**网格形式**展示所有模块入口：

- **两列布局**，每列模块数量尽量均衡
- 每个模块项：图标（32px）+ 名称 + 使用频次标签
- 当前所在模块项高亮显示（左侧品牌色边框 `#4F46E5`）
- 支持键盘上下导航（方向键选择，Enter确认）

### 4.3 弹窗内快捷操作

- **排列切换开关**：弹窗右上角提供"最常用在左/在最右"切换
- 切换即时生效，并同步更新本地缓存
- 切换后模块卡片整体滑动重排（0.3s动画）

### 4.4 快速搜索（可选）

- 弹窗顶部提供搜索框，支持模糊匹配模块名称
- 输入时实时过滤，过滤结果高亮匹配文字
- 空搜索结果时显示"未找到相关模块"

## 五、模块点击行为

### 5.1 点击模块

1. 更新该模块的usageStats（+1）
2. 触发页面跳转至对应模块首页
3. 若当前有未保存的表单数据，弹出确认提示

### 5.2 快捷键支持

| 快捷键 | 功能 |
|---|---|
| `M` 或 `/` | 打开模块入口弹窗 |
| `Esc` | 关闭弹窗 |
| `↑` / `↓` | 在弹窗内上下切换模块 |
| `Enter` | 进入选中模块 |
| `1-5` | 直接进入第N个模块 |

## 六、响应式适配

### 6.1 断点

| 屏幕宽度 | 模块入口页布局 |
|---|---|
| ≥1440px | 4列热力图 |
| 1024px ~ 1439px | 3列热力图 |
| <1024px | 2列热力图，弹窗宽度自适应屏幕 |

### 6.2 移动端

- 模块入口页在移动端（<768px）显示为**单列滚动列表**
- "更多"按钮保留，弹窗改为全屏底部抽屉样式
- 热力图布局在移动端不适用，改为默认列表排序（按频次）

## 七、技术实现建议

### 7.1 前端框架

- 使用现有前端框架（建议 Vue 3 或 React 18）
- CSS动画优先使用CSS Transition/Animation，复杂动画可引入 GSAP
- 热力图布局算法：按usageStats排序后，按宽度权重分配

### 7.2 状态管理

- 模块入口相关状态（偏好、频次）使用 `localStorage` + `Zustand` / `Pinia` 持久化
- 弹窗开启/关闭状态使用局部组件状态

### 7.3 性能

- 模块入口页骨架屏加载，避免白屏
- 动效使用 `will-change` 和 `transform`，避免重排
- usageStats更新使用防抖（debounce 500ms）合并写入

## 九、验收标准

| 编号 | 验收条件 |
|---|---|
| AC-01 | 模块入口页加载后，模块卡片依次动画淡入 |
| AC-02 | 鼠标hover卡片时，卡片上浮且有发光动效 |
| AC-03 | 点击模块后，usageStats +1，且下次访问时卡片大小正确反映频次 |
| AC-04 | 切换排列方向后，页面立即响应，卡片平滑滑动 |
| AC-05 | 刷新页面后，排列方向和频次数据不丢失 |
| AC-06 | 点击顶部"更多"按钮，弹窗从按钮位置向下展开 |
| AC-07 | 弹窗内显示所有模块，当前模块高亮 |
| AC-08 | 按Esc或点击遮罩可关闭弹窗 |
| AC-09 | 键盘快捷键M打开弹窗，方向键导航，Enter确认 |
| AC-10 | 移动端（<768px）显示单列列表布局 |
